# Глубины

Карты и отряд для **Pathfinder 2e**: генератор подземелий, жетоны, туман войны, аккаунты мастера и игроков, сохранение в SQLite.

## Требования

- [Node.js](https://nodejs.org/) 22+ (используется встроенный модуль `node:sqlite`)

## Запуск

```bash
npm start
```

Откройте в браузере: [http://127.0.0.1:8765](http://127.0.0.1:8765)

> Не открывайте `index.html` как `file://` — нужен сервер, иначе API и база не работают.

Режим разработки с автоперезапуском:

```bash
npm run dev
```

## Возможности

- **Карта** — генерация лабиринта (лес / город / подземелье), сид, вид сверху и изометрия
- **Туман войны** — режим мастера (вся карта) и игрока (посещённые комнаты + peek)
- **Комнаты** — мастер разрешает вход кнопкой «Вход»; игроки не проходят дальше без разрешения
- **Отряд** — герои и враги, бестиарий PF2e, профили персонажей
- **Аккаунты** — мастер правит всё; игрок ходит своим героем (стрелки / WASD), смотрит листы пати
- **Сохранение** — карта, персонажи, враги и аккаунты в `data/glubiny.sqlite` (`F5` сохраняет карту)

## Управление

| Клавиша | Действие |
|--------|----------|
| `WASD` / стрелки | Ход выбранного жетона |
| `Space` | Снять выделение |
| `Esc` | Назад / закрыть комнату |
| `1` / `2` | Жетоны / препятствия (мастер) |
| `Q` / `E` | Поворот в изометрии |
| Колёсико | Масштаб |
| Средняя кнопка мыши | Сдвиг карты |
| `F5` | Сохранить карту и обновить страницу |

Игроки **не** переставляют жетон кликом по клетке — только стрелками / WASD. Генерация карты доступна только мастеру.

## Структура

```
├── index.html          # UI
├── css/styles.css
├── js/                 # клиент (карта, отряд, аккаунты)
├── server/             # Express-подобный HTTP + SQLite
├── data/               # bestiary.json; sqlite создаётся при запуске
└── scripts/            # сборка бестиария
```

База `data/glubiny.sqlite` в `.gitignore` и не коммитится.
