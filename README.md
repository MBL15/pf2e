# Глубины

Веб-приложение для настольных сессий **Pathfinder 2e**: генератор подземелий, карта с туманом войны, жетоны героев и врагов, профили персонажей и аккаунты мастера с игроками. Данные сессии (карта, отряд, аккаунты) хранятся в SQLite на сервере.

## Развёртывание

**Требования:** Python 3.10+

```bash
git clone https://github.com/MBL15/pf2e.git
cd pf2e
python -m server.main
```

Откройте в браузере: [http://127.0.0.1:8765](http://127.0.0.1:8765)

> Не открывайте `index.html` напрямую (`file://`) — без сервера API и база не работают.

При первом запуске создаётся `data/glubiny.sqlite`. Порт можно задать переменной окружения `PORT` (по умолчанию `8765`).
